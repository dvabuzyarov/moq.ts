    The default mock options that would applied to all instantiating Mock objects. By default it sets IMockOptions.target as a function, IMockOptions.injectorConfig as instance of DefaultInjectorConfig and IMockOptions.name as undefined. If an options are passed as constructor parameter Mock.constructor they will override the default options.
